Project Idea

A great project idea would be to develop or help develop the next big app. As for what app I would like to develop a study app. The project would include a study app that is compatible with iOS, Android and PC. The study app will include many different components and will either be very cheap or free. The app will also have a simple user interface which will give off a welcoming vibe to all users. The app design would be the easier part of the app compared to the programming component as my programming skills aren’t very furnished yet. This project is very interesting as there are very few ‘great’ study apps out there nowadays and technology is the way of the future. While studying through high school I always wished that there was an app that would help me effectively take notes, keep organised and motivate me to study more. Creating a study app that is great and cheap will be the saving grace for many students in our society today.

There are many different features that my application will have and many different services it may provide. Firstly, I would like the basic concept of note taking. Similar to other apps out there my app would allow the user to take notes where ever they please, however they please. I would like to implement a camera for those who like to hand write notes, so they can take a snapshot of their notes and the app will convert the handwriting to text allowing the user to use the Find/Search function to look-up keywords quickly and efficiently. I would also like the app to enable attachments, so the user may attach handouts, downloaded files, etc… Secondly, the app will have to have cloud saving capabilities. This is a very important concept nowadays because it is very efficient to have your notes/study material whenever you go, as long as you have connection to the internet. For this to work I would like to have a decently sized cloud storage as many apps that provide this have very limited storage capabilities. Thirdly, to complement the last point I would love to have download capabilities or autosave to the cloud capabilities. This would mean that if you are travelling places where you know there isn’t internet connection you can download your notes you have so far. This would allow you to continue to work/add on notes to what you have and the application would automatically save the extra work you have done after you have reconnected to the internet. Finally the application will provide a to-do timer/list. This is where the user may input when things are due, what they have to do after they get home, etc… This may act like an alarm or like a calendar notification and help the user be very time efficient. As for the services my app would have, I would firstly allow the user to search keywords and cross reference them to any online books/sources that may help the user in their studies. This would also auto create a bibliography/reference list in any format of their choosing. Another service my application would provide is the service to connect to a free tutor or another student using the app if the user may need help with a certain keyword/topic. This service would also have an appointment booking where a user may book a session with another student to discuss topics or to ask for help. Another service the application may have is a service which provides the user with an assignment/essay check. This service would greatly help the user as they will get feedback before submitting their assignment, so they may improve their work and get optimal marks. Finally, the application will either be free or cheap <$10. This is because a price tag on applications is a big turn off for many students as they do not have much money to spend and if they do they wouldn’t not spend it on a study application. If the application were to be free it would also have a donation button to keep the application up to date similar to that of Wikipedia.

There are many applications out there which may allow the creation/development of apps. Appery.io, Mobile Roadie, AppMachine and AppBuilder are examples of many software tools that may be used to create my app. Researching languages, BuildFire.js, Python and Java are the top three programming languages which are used for applications followed by the likes of PHP and C++. All these components together with a good plan, motivation and a creative mind will allow an app like this to be created.

The development of this application will require many different skills and very precise and effective planning. Programming skills will be required and will be learnt through the IT Course at RMIT University. Creative skills will have to be applied for an optimal app production as many concepts such as User Interface has to be looked at closely and provide what is best and wanted from users. Great research techniques have to be implemented as comparison with competitors and price of production has to be taken into account when investing in the project. Finally, teamwork skills may have to be used as creating such a big global app will require more than one person for time efficiency.

If the project ends up being a success, the outcome will solve many student’s problems with apps to use. There are very few applications out there that give students what they need so they must use multiple apps. With this one app, everything is in one so students will just have one simple app they can use. This may impact the way students study in the future and help optimise their time and help them understand what they are doing at all times.
